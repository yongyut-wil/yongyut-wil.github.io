## Live CV Website

A professional and modern CV website built with HTML, Tailwind CSS, and JavaScript. Features responsive design, print optimization, and smooth animations.

### Features

- 🎨 **Modern Design**: Clean and professional layout with Tailwind CSS
- 📱 **Responsive**: Works perfectly on desktop and mobile devices
- 🖨️ **Print Optimized**: Clean PDF export without browser headers/footers
- ✨ **Interactive Elements**: Smooth hover animations and transitions
- 🎯 **Organized Content**: Well-structured sections for skills, experience, and projects
